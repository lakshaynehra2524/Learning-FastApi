# 🚀 Learning-FastAPI

A practical repository documenting my end-to-end learning journey with **FastAPI** through code.

The purpose of this repository is to learn FastAPI by building real applications, experimenting with its features, and implementing best practices. Rather than focusing on theory, every topic is explored through hands-on coding examples and mini-projects.

## 🎯 Objectives

* Learn FastAPI from beginner to advanced.
* Build production-oriented REST APIs.
* Understand FastAPI's internal workflows and features.
* Maintain a structured record of my learning journey.
* Create reusable code examples for future reference.

> More topics may be added as my learning progresses.

## 🌿 Repository Organization

This repository is organized using **Git branches**.

Each tutorial or learning milestone is maintained in its own branch instead of being merged into `main`. This keeps every chapter independent and preserves the progression of my learning.

Example:

* `Tutorial-01`
* `Tutorial-02`
* `Tutorial-03`
* ...

Use the **Branch** selector on GitHub to explore a specific tutorial.
